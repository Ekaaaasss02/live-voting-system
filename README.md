# Live Voting System

## Project Description

The Live Voting System is a web-based application that allows users to vote in real-time for different options. The system records votes in a database and displays results instantly.

This project is developed using **Python Flask, HTML, CSS, and SQLite**.

## Features

* User login system
* Live voting functionality
* Vote storage using SQLite database
* Real-time vote counting
* Simple and responsive user interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SQLite

## How to Run the Project

1. Install Python
2. Install Flask

```
pip install flask
```

3. Run the application

```
python app.py
```

4. Open browser and go to

```
http://127.0.0.1:5000
```
